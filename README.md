## Installation
adjust to the nodejs package manager commonly used
- Install packages

```bash
# yarn
yarn install

# npm    
npm install
```


- Development: app run in http://localhost:3000
```bash
# yarn
yarn dev

# npm    
npm run dev
```

- Build for production, folder location: `dist` 
```bash
# yarn
yarn build

# npm
npm run build
```

- Serve folder `dist`. check http://localhost:5000 
```bash
# yarn
yarn serve

# npm
npm run build
```
